# Sprites grandes de entrenadores (batalla)

Formato: PNG con fondo transparente (RGBA).
Tamaño sugerido: ~200-400px de alto.
Nombrar según el ID del NPC: alejandro.png, alessandro.png, etc.

El módulo trainer-big-sprites.js buscará aquí primero;
si no encuentra el PNG, usará el dibujo pixel-art de respaldo.
